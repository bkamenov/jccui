JCC class properties


  • jcc.options

    A set of various options which can be set before the call of 'jcc.init()';

    • jcc.options.alert.theme

      The theme to be used for the alert dialog box as a string - "a" or "b". Default: undefined - will use the theme from the HTML body if set or theme "a" otherwise.

    • jcc.options.alert.defaultTitle

      The string to be used as a default dialog title. Default: "Message:"

    • jcc.options.alert.okText

      The string to be used for the 'OK' button. Default: "OK"


    • jcc.options.prompt.theme

      The theme to be used for the prompt dialog box as a string - "a" or "b". Default: undefined - will use the theme from the HTML body if set or theme "a" otherwise.

    • jcc.options.prompt.defaultTitle

      The string to be used as a default dialog title. Default: "Input needed:"

    • jcc.options.prompt.okText

      The string to be used for the 'OK' button. Default: "OK"

    • jcc.options.prompt.cancelText

      The string to be used for the 'Cancel' button. Default: "Cancel"


    • jcc.options.confirm.theme

      The theme to be used for the confirm dialog box as a string - "a" or "b". Default: undefined - will use the theme from the HTML body if set or theme "a" otherwise.

    • jcc.options.confirm.defaultTitle

      The string to be used as a default dialog title. Default: "Confirmation needed:"

    • jcc.options.confirm.yesText

      The string to be used for the 'Yes' button. Default: "Yes"

    • jcc.options.confirm.noText

      The string to be used for the 'No' button. Default: "No"


    • jcc.options.datetime.theme

      The theme to be used for the date time picker dialog box as a string - "a" or "b". Default: undefined - will use the theme from the HTML body if set or theme "a" otherwise.

    • jcc.options.datetime.okText

      The string to be used for the 'OK' button. Default: "OK"

    • jcc.options.datetime.cancelText

      The string to be used for the 'Cancel' button. Default: "Cancel"

    • jcc.options.datetime.nowText

      The string to be used for the 'Now' button. Default: "Now"

    • jcc.options.datetime.hourText

      The 'hour' identifier to be used for the hours combo box. Default: "h"

    • jcc.options.datetime.minuteText

      The 'minute' identifier to be used for the minutes combo box. Default: "min"

    • jcc.options.datetime.months

      Array of strings with the months' names. Default: ["January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"]

    • jcc.options.datetime.weekDays

      Array of strings with the week days' short names. Default: ["Su", "Mo", "Tu", "We", "Th", "Fr", "Sa"]

    • jcc.options.datetime.DIALOG_MODE.DATE

      Flagged value used for jcc.datetime method's argument 'mode'. It involves a date picking.

    • jcc.options.datetime.DIALOG_MODE.TIME

      Flagged value used for jcc.datetime method's argument 'mode'. It involves time picking.

    • jcc.options.datetime.overrideDefaultInputType

      A boolean if the default browser built-in dialogs for the input types [date, time, datetime, datetime-local] should be overridden. Default: false.


    • jcc.options.colorpicker.theme

      The theme to be used for the color picker dialog box as a string - "a" or "b". Default: undefined - will use the theme from the HTML body if set or theme "a" otherwise.

    • jcc.options.colorpicker.okText

      The string to be used for the 'OK' button. Default: "OK"

    • jcc.options.colorpicker.cancelText

      The string to be used for the 'Cancel' button. Default: "Cancel"

    • jcc.options.colorpicker.COLOR_FORMAT.HEX

      Color mode value used for jcc.colorpicker method's argument 'mode'. The element value and background color are set as a hex value (e.g. #aabbcc).

    • jcc.options.colorpicker.COLOR_FORMAT.RGB

      Color mode value used for jcc.colorpicker method's argument 'mode'. The element value and background color are set as RGB value (e.g. rgb(123,123,255)).

    • jcc.options.colorpicker.COLOR_FORMAT.RGBA

      Color mode value used for jcc.colorpicker method's argument 'mode'. The element value and background color are set as RGBA value (e.g. rgba(123,123,255, 0.9)). Using the RGBA mode involves alpha channel selection within the color picker dialog.

    • jcc.options.colorpicker.overrideDefaultInputType

      A boolean if the default browser built-in dialog for the input type 'color' should be overridden. Default: false.

  • jcc.resizeMonitor

    An object used to monitor HTML-elements for size changes. Each registered HTML-element will be fired a 'j-resized' event once its size has changed. Please see the JCC class methods 'jcc.resizeMonitor.addElement' and 'jcc.resizeMonitor.removeElement' for more information about adding/removing HTML-elements to/from the resize monitor.
