Collapsible set widget


The collapsible set widget unites two or more collapsible widgets into one single accordeon-like widget. 


<div data-role="collapsible-set" data-corners="smooth">
			
	<div data-role="collapsible">
		<label>
			<input type="checkbox"/>
			<i data-role="icon" data-icon="ion-umbrella"></i>
			<span>Collapsible</span>
		</label>
		<div data-role="container">
			<h1>1. Collapsible contents</h1>
			<p>Some content is supposed to come here.</p>
			
			<h2>1.1. Collapsible contents continued</h2>
			<p>Some continued content is supposed to come here.</p>
		</div>
	</div>

All options are accessible throught the getOption and setOptions method of the widget [topmostMarkupElement.collapsibleSet] object. After setting any of the options for the collapsible set widget, it will be unenhanced and enhanced to apply the newly set options.

Please use the theme method of the widget to read the current theme.

  • theme

    HTML-Markup notation: data-theme

    JavaScript notation: theme


    The theme name as string to be used for the widget. Possible values 'a' or 'b'. By default, 'a' stays for dark theme, while 'b' for the bright theme. If omitted, the widget will inherit the theme from its parent.

  • corners

    HTML-Markup notation: data-corners

    JavaScript notation: corners


    Possible values:

    • flat

      The corners of the widget will be rectangular.

    • smooth

      The corners of the widget will be smoothed.


    Default is: smooth
