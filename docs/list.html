List widget


A visual representation of an item list. It can contain sections. The items can be clickable or not. Within the items additional widgets can be added.

This documentation will describe also the list-item widget which is an integral part of the list widget. 


<!-- Basic list instance -->
<ul data-role="list">
	<li data-role="list-item" data-isotope="header">
		<span>Header item</span>
	</li>
	<li data-role="list-item" data-isotope="chevron" data-tap="yes">
		<span>Item 1</span>
	</li>
	<li data-role="list-item" data-isotope="chevron" data-tap="yes">
		<span>Item 2</span>
	</li>
</ul>
<!-- An inset list instance -->
<ul data-role="list" data-isotope="inset" data-corners="smooth">
	<li data-role="list-item" data-isotope="header">
		<span>Header item</span>
	</li>
	<li data-role="list-item">
		<span>Item 1</span>
	</li>
	<li data-role="list-item">
		<span>Item 2</span>
	</li>
</ul>

All options are accessible throught the getOption and setOptions method of the widget [topmostMarkupElement.list] object or the listItemMarkupElement.listItem object. After setting any of the options for the list widget, it will be unenhanced and enhanced to apply the newly set options.

Please use the theme method of the widget to read the current theme.

  • [list] corners

    HTML-Markup notation: data-corners

    JavaScript notation: corners


    Possible values:

    • flat

      The corners of the widget will be rectangular.

    • smooth

      The corners of the widget will be smoothed.


    Default is: smooth

    The corners property has no meaning if the isotope property is set to normal.

  • [list] isotope

    HTML-Markup notation: data-isotope

    JavaScript notation: isotope


    Possible values:

    • normal

      A list with a flat border looking like a separate unit.

    • inset

      A simple list without border - intended for embedding.

    Default is: normal

  • [list-item] isotope

    HTML-Markup notation: data-isotope

    JavaScript notation: isotope


    Possible values:

    • normal

      Ordinary list item.

    • chevron

      A list item with a chevron on the right.

    • header

      A list item intended to be a header. It has larger font and a thicker underline border.

    Default is: normal

  • [list-item] tap

    HTML-Markup notation: data-tap

    JavaScript notation: tap


    If set to yes, the list item will be highlighted when tapped and a pointer cursor will be set if moved over the list item with the mouse.

    Default is: no

  • [list-item] h-dense

    HTML-Markup notation: data-h-dense

    JavaScript notation: hDense


    If set to yes, the list item will have no horizontal padding.

    Default is: no

  • [list-item] v-dense

    HTML-Markup notation: data-v-dense

    JavaScript notation: vDense


    If set to yes, the list item will have no vertical padding.

    Default is: no

  • [list-item] dense

    HTML-Markup notation: data-dense

    JavaScript notation: dense


    If set to yes, the list item will have no horizontal and vertical padding.

    Default is: no

  • [list and list-item] theme

    HTML-Markup notation: data-theme

    JavaScript notation: theme


    The theme name as string to be used for the widget. Possible values 'a' or 'b'. By default, 'a' stays for dark theme, while 'b' for the bright theme. If omitted, the widget will inherit the theme from its parent.
