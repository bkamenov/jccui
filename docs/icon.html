Icon widget


The icon widget provides JCC with the abilty to display web-font icons and to integrate them into other widgets.

You can apply colors, animation and rotation to each icon widget instance. 


<i data-role="icon" data-icon="ion-umbrella" style="font-size:20px;"></i>
<i data-role="icon" data-rotate="45" data-icon="ion-umbrella" style="font-size:40px;"></i>
<i data-role="icon" data-flip="horizontal" data-icon="ion-umbrella" style="font-size:20px; color:yellow;"></i>
<i data-role="icon" data-flip="vertical"  data-icon="ion-umbrella" style="font-size:20px;"></i>
<i data-role="icon" data-icon="ion-umbrella" data-animation="pulse" style="font-size:20px;"></i>
<i data-role="icon" data-icon="ion-umbrella" data-animation="spin" style="font-size:20px;"></i>
<i data-role="icon" data-icon="ion-umbrella" data-animation="fade" style="font-size:20px;"></i>

All options are accessible throught the getOption and setOptions method of the widget [iconMarkupElement.icon] object. After setting any of the options for the icon widget, it will be unenhanced and enhanced to apply the newly set options.

Please use the theme method of the widget to read the current theme.

  • theme

    HTML-Markup notation: data-theme

    JavaScript notation: theme


    The theme name as string to be used for the widget. Possible values 'a' or 'b'. By default, 'a' stays for dark theme, while 'b' for the bright theme. If omitted, the widget will inherit the theme from its parent.

  • rotate

    HTML-Markup notation: data-rotate

    JavaScript notation: rotate


    Specifies the clockwise angle of rotation in degrees.

    Default is: empty string

    Remarks: If not an empty string, it will force the widget to ignore the flip option.

  • flip

    HTML-Markup notation: data-flip

    JavaScript notation: flip


    Specifies if the icon should be flipped horizontally or vertically. Allowed values are:

    • none

      No flip at all.

    • horizontal

      Horizontal fliping.

    • vertical

      Vertical fliping.


    Default is: none.

  • icon

    HTML-Markup notation: data-icon

    JavaScript notation: icon


    Specifies the icon as specified in the web font CSS.

    Default is: empty string

  • animation

    HTML-Markup notation: data-animation

    JavaScript notation: animation


    Specifies the animation to be applied for the icon. Allowed values are:

    • none

      No animation.

    • spin

      The icon will spin.

    • fade

      A pulse-fading icon.

    • pulse

      A pulse-scaling icon.

    Default is: none.

  • position

    HTML-Markup notation: data-position

    JavaScript notation: position


    Specifies the position of the icon widget when it is used within other widgets and elements. Allowed values are:

    • auto

      No special position is applied.

    • left

      The icon is forced to be on the left.

    • right

      The icon is forced to be on the right.

    Default is: auto.
