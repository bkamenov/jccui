Sample usage


To make JCC work you need a simple HTML to start (see the snippet bellow). The main point is to include the 'jcc.min.js' and 'jcc.min.css' in your HTML and then at the end of your HTML to initialize the JCCUI library with the DOM-node you want (i.e. document.body). Please note that we have marked the body to have visibility hidden and after initializing the JCCUI we set it back to visible. This optional trick might be needed to avoid showing the 'unenhanced' version of your HTML-code especialy when the HTML-code is really 'heavy'. 


<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no, minimum-scale=1.0, maximum-scale=1.0" />
		
		<!-- For IE on WP8 and WP8.1 equivalent of CSS: -webkit-tap-highlight-color: rgba(0,0,0,0); -->
		<meta name="msapplication-tap-highlight" content="no"/>
		
		<title>Your page title</title>
		
		<link rel="stylesheet" href="../ionicons/ionicons.css">
		<link rel="stylesheet" href="../dist/jcc.min.css">
		<script src="../dist/jcc.min.js"></script>
		
	</head>

	<body data-role="container" data-theme="a" style="visibility:hidden;">
		<h1>Collapsible example</h1>
		<br/>
		
		<div data-role="collapsible" data-expanded="yes">
			<label>
				<input type="checkbox"/>
				<span>Collapsible</span>
			</label>
			<div data-role="container">
				<h1>1. Collapsible contents</h1>
				<p>Some content is supposed to come here.</p>
				
				<h2>1.1. Collapsible contents continued</h2>
				<p>Some continued content is supposed to come here.</p>
			</div>
		</div>
		
	</body>
	
	<script type="text/javascript">
			
		//JCC internals initialization + Enhancement of the DOM hierarchy [you can place any DOM node here you wish to enhance]
		jcc.init(document.body);
		document.body.style.visibility = "visible";

	</script>

</html>
