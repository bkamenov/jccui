Delimiter widget


A horizontal or vertical bar to delimit different content from each other. 


<!--Just a horizontal bar-->
<div data-role="delimiter" data-thickness="1" data-orientation="horizontal"></div>
<!--Gradient both sides fading horizontal bar-->
<div data-role="delimiter" data-colors="rgba(0,0,0,0),rgba(149,165,166,1),rgba(0,0,0,0)" data-thickness="1" data-orientation="horizontal"></div>
<!--Gradient both sides fading vertical bar-->
<div data-role="delimiter" data-colors="rgba(0,0,0,0),rgba(149,165,166,1),rgba(0,0,0,0)" data-thickness="1" data-orientation="vertical"></div>

All options are accessible throught the getOption and setOptions method of the widget [topmostMarkupElement.delimiter] object. After setting any of the options for the delimiter widget, it will be unenhanced and enhanced to apply the newly set options.

Please use the theme method of the widget to read the current theme.

  • theme

    HTML-Markup notation: data-theme

    JavaScript notation: theme


    The theme name as string to be used for the widget. Possible values 'a' or 'b'. By default, 'a' stays for dark theme, while 'b' for the bright theme. If omitted, the widget will inherit the theme from its parent.

  • thickness

    HTML-Markup notation: data-thickness

    JavaScript notation: thickness


    The thickness of the delimiter bar in pixels.

    Default is: 1.

  • colors

    HTML-Markup notation: data-colors

    JavaScript notation: colors


    The colors through which the gradient bar should go from left to right. Acceptable value is a comma-separated list of CSS rgb(r, g, b) and rgba(r, g, b, a) sub-values e.g. rgb(149,165,166) and rgba(149,165,166,1) respectively.

    Default is: empty string.
