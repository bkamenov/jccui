Dropdown widget


A JCC styled version of the original HTML-dropdown element known as select.

It supports icons but only on the left side. For more information, see the icon widget reference. 


<!-- Simple dropdown -->
<div data-role="dropdown">
	<select>
		<option selected>Select Box</option>
		<option>Short Option</option>
		<option>This Is A Longer Option</option>
	</select>
</div>
<!-- Disabled dropdown -->
<div data-role="dropdown">
	<select disabled>
		<option selected>Select Box</option>
		<option>Short Option</option>
		<option>This Is A Longer Option</option>
	</select>
</div>
<!-- Dropdown with icon -->
<div data-role="dropdown">
	<select>
		<option selected>Select Box</option>
		<option>Short Option</option>
		<option>This Is A Longer Option</option>
	</select>
	<i data-role="icon" data-icon="ion-umbrella"></i>
</div>

All options are accessible throught the getOption and setOptions method of the widget [topmostMarkupElement.dropdown] object. After setting any of the options for the dropdown widget, it will be unenhanced and enhanced to apply the newly set options.

Please use the theme method of the widget to read the current theme.

  • theme

    HTML-Markup notation: data-theme

    JavaScript notation: theme


    The theme name as string to be used for the widget. Possible values 'a' or 'b'. By default, 'a' stays for dark theme, while 'b' for the bright theme. If omitted, the widget will inherit the theme from its parent.

  • corners

    HTML-Markup notation: data-corners

    JavaScript notation: corners


    Possible values:

    • flat

      The corners of the widget will be rectangular.

    • smooth

      The corners of the widget will be smoothed.

    • round

      The corners of the widget will be rounded.


    Default is: smooth


topmostMarkupElement.dropdown.disabled(disabled)

Returns/sets the disabled state of the dropdown.

Arguments:

  • [optional] disabled

    If omitted, the method will return the disabled state of the dropdown widget. If true is provided, the dropdown will be set as disabled.

topmostMarkupElement.dropdown.selectedIndex(selectedIndex)

Returns/sets the selected item's index of the dropdown.

Arguments:

  • [optional] selectedIndex

    If omitted, the method will return the selected item's index of the dropdown widget. If provided, the dropdown will select the item whose index is provided or lose the selection if negative value is provided.

The dropdown widget supports all HTML-events supported by the select tag and all event handlers should be applied to the select element.
