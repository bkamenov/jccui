Flipswitch widget


A typical implementation of a checkbox having the flip-switch view. 


<label data-role="flipswitch">
	<input type="checkbox" checked />
</label>

<label data-role="flipswitch">
	<input type="checkbox" checked disabled />
</label>

All options are accessible throught the getOption and setOptions method of the widget [topmostMarkupElement.flipswitch] object. After setting any of the options for the flipswitch widget, it will be unenhanced and enhanced to apply the newly set options.

Please use the theme method of the widget to read the current theme.

  • theme

    HTML-Markup notation: data-theme

    JavaScript notation: theme


    The theme name as string to be used for the widget. Possible values 'a' or 'b'. By default, 'a' stays for dark theme, while 'b' for the bright theme. If omitted, the widget will inherit the theme from its parent.


topmostMarkupElement.flipswitch.checked(newState)

If provided with an argument it sets the new checked state of the flipswitch. Otherwise, it just returns the current checked state of the flipswitch.

Arguments:

  • [optional] newState

    A boolean value to set the flipswitch checked state.

topmostMarkupElement.flipswitch.disabled(newState)

If provided with an argument it sets the new disabled state of the flipswitch. Otherwise, it just returns the current disabled state of the flipswitch.

Arguments:

  • [optional] newState

    A boolean value to set the flipswitch disabled state.

Flipswitch widget supports all the default HTML events. The handlers should be applied to the input HTML-element of the markup.
