Container example

Container widget is not really a widget but just a definition of a BODY or DIV to have a theme based font and background colors and padding. It also includes definition for headings (H1 to H6), paragraphs (P), labels (LABEL) and links (A). Another important feature of container is that the cursor is set to default, selection is disabled within its elements except inputs (INPUT and TEXTAREA). Browser enhancements for the elements within a container are also disabled.

All widgets should be placed within a container but this is NOT a must - widgets outside a container must have set EXPLICITLY the styles disabling the selection and browser enhancements.
